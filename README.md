## Requirements

- Windows 10 Version 1903 or later

## Usage

1. Install [latest version](https://www.microsoft.com/store/apps/9njmtkk6jc0q)
2. Launch WinQuickLook
3. Open explorer
4. Select a file
5. Press a space-key
